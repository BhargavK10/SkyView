SkyView is an open source project that aims to assist begineers to Astronomy and star gazing!
SkyView provides a good overview of the celestial objects that will be visible in the sky tonight and can be viewed with nakaed eyes or basic telescopes! The Get Started page also teaches begineers how to get started with astronomy/ star gazing/ Astrophotography as a hobby. This is an open source page meaning anyone can contribute to the development and upkeep of this page, click on the Github icon on the bottom of the sidebar to get the source code, and while you are there, consider supporting us on Instagram and Youtube :)
